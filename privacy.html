Политика конфиденциальности

SAKURA обрабатывает персональные данные пользователя для регистрации, оформления доставки, хранения истории заказов и связи по заказу.

Какие данные используются

Имя, email, телефон, адрес доставки, состав заказов и технические данные, необходимые для работы сайта.

Для чего нужны данные

Данные используются для создания аккаунта, подтверждения заказа, передачи заказа администратору и отображения истории покупок в личном кабинете.

Хранение данных

Данные хранятся в базе Supabase. Доступ ограничивается правилами безопасности базы данных.

Права пользователя

Пользователь может изменить контактные данные в личном кабинете или обратиться к администратору для удаления данных.
